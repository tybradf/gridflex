GRIDFLEX // PJM live
GridFlex is a live, automated system that forecasts PJM electricity demand and carbon intensity, then calculates what it's worth to shift flexible power use to cleaner hours. The forecast above is currently beating PJM's own day-ahead model. Every shift recommendation below passes a statistical significance check — not a guess. Built solo, zero-cost infrastructure, fully open source.
Zone-level demand and system-wide fuel-mix data publish on independent schedules — the carbon scrubber may lag "now" by up to ~24h while one data source catches up to the other. Code & methodology →
